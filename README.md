# Hamming-Code-7-4-for-single-bit-error-detection-and-correction.
A digital logic based implementation of the Hamming (7,4) error correction code capable of detecting and correcting single-bit errors during data transmission. The project includes encoder, parity generation, syndrome calculation, and correction logic.
